## Tara Development Portfolio

The "Tara" development is a collection of **five interconnected projects** owned by Tara Forest, LLC. These must be tracked as a portfolio, not individually.

### Project Summary

| Project | Size | Units | Status | Purpose |
|---------|------|-------|--------|---------|
| Tara Forest West | 395 acres | 540 lots | Pending | Residential |
| Tara Baywood | 36 acres | 211 townhomes | Under construction | Residential |
| Tara Forest East | 148 acres | 340 lots | Under construction | Residential |
| Tara April | N/A | N/A | Awaiting final plat | Stormwater infrastructure |
| Tara Phoenicia | TBD | Mixed-use | Awaiting final plat | Commercial/Residential |

**Total: ~580 acres, 1,000+ residential units**

### Key Application
- **Tara April Special Exception:** Application #PSE22-0002
- **Parcel:** Tax Parcel Number 03020-000-000
- **Location:** North of US 441, east of I-75
- **Request:** Special Exception to allow stormwater facilities on Agricultural (A) zoned land
